# Taxonomy Web App

Next.js frontend for the taxonomy platform. The app talks to `api-service` and derives its data contracts from the backend DTOs and controller routes.

## Environment

Create `web-app/.env.local`:

```bash
TAXONOMY_API_BASE_URL=http://localhost:8080
TAXONOMY_API_KEY=dev-api-key-change-me
```

## Scripts

```bash
npm install
npm run dev
```
