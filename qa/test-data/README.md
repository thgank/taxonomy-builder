# Test Data Notes

Keep only small, sanitized, non-sensitive sample files in this directory.

Recommended samples:

- `sample-en.txt`
- `sample-ru.txt`
- `sample-policy.html`
- `sample-report.pdf`
- `sample-article.docx`

Guidelines:

- prefer files smaller than 1 MB
- avoid copyrighted or sensitive content
- keep document topics aligned with a single domain so taxonomy output is easier
  to validate
- if you add files here, document their expected behavior in the manual
  checklist or a future regression note
